# Learn By Make Website 🌐

**Official website for Learn By Make - Proprietary technology education curriculum**

## 🚀 Website

👉 **Live website**: [https://theterminalguyYT.github.io/learnbymake-website/](https://theterminalguyYT.github.io/learnbymake-website/)

## 📁 Project Structure

```
learnbymake-website/
├── index.html          # Main website homepage
├── css/
│   └── style.css      # Website styles
├── js/
│   └── main.js        # JavaScript functionality
├── pages/
│   ├── curriculum.html # Detailed curriculum information
│   ├── about.html      # About Learn By Make
│   └── contact.html    # Contact information
├── images/
│   └── favicon.ico     # Website icon
├── 404.html            # Custom 404 error page
└── README.md           # This file
```

## 🛠️ Technologies Used

- **HTML5** - Semantic structure for accessibility
- **CSS3** - Modern styling with CSS variables
- **JavaScript** - Interactive elements and animations
- **Responsive Design** - Mobile-first approach
- **GitHub Pages** - Website hosting

## 📱 Features

✅ **Fully Responsive** - Works on all devices (mobile, tablet, desktop)
✅ **Modern UI** - Clean, professional design with gradient accents
✅ **Fast Loading** - Optimized assets and efficient code
✅ **Accessible** - Semantic HTML, proper contrast, and ARIA labels
✅ **Interactive** - Smooth animations, hover effects, and transitions
✅ **Multi-Page** - Comprehensive site structure with dedicated pages
✅ **Custom 404** - Friendly error page for broken links

## 🎨 Design Highlights

- **Color Scheme**: Blue/purple gradient (#4a6bff to #6a5acd) with red accent (#ff6b6b)
- **Typography**: 'Segoe UI' font family for clean readability
- **Layout**: CSS Grid and Flexbox for responsive design
- **Animations**: Subtle fade-in and hover effects
- **Components**: Reusable card system, buttons, and navigation

## 🚀 Development

### Local Preview

1. Clone this repository (private access required)
2. Open `index.html` in any modern browser
3. Test responsive design at different screen sizes

### Deployment

The website automatically deploys to GitHub Pages when changes are pushed to the `master` branch.
