# OpenSystemClasses Website 🌐

This repository contains the official website for OpenSystemClasses - a hands-on technology education curriculum for young makers.

## 🚀 Website

👉 **Visit the live website**: [https://theterminalguyYT.github.io/opensystemclasses-website/](https://theterminalguyYT.github.io/opensystemclasses-website/)

## 📁 Project Structure

```
opensystemclasses-website/
├── index.html          # Main website (single-page design)
├── README.md           # This file
└── assets/             # (Future) Images, CSS, JS files
```

## 🛠️ Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Modern styling with CSS variables
- **JavaScript** - Interactive elements
- **Responsive Design** - Works on all devices
- **GitHub Pages** - Free hosting

## 📱 Features

✅ **Fully Responsive** - Works on mobile, tablet, and desktop
✅ **Modern UI** - Clean, professional design
✅ **Fast Loading** - Single HTML file, no external dependencies
✅ **Accessible** - Semantic HTML and good contrast
✅ **Interactive** - Hover effects and smooth scrolling

## 🎨 Design Highlights

- **Color Scheme**: Blue/purple gradient with accent colors
- **Typography**: Clean, readable fonts
- **Layout**: Grid-based responsive design
- **Animations**: Subtle hover effects and transitions
- **Navigation**: Sticky header with smooth scrolling

## 🚀 How to Use

### For Development:

1. Clone this repository:
   ```bash
   git clone https://github.com/theterminalguyYT/opensystemclasses-website.git
   ```

2. Open `index.html` in your browser to preview

3. Make changes and push to update the live site

### For Deployment:

The site automatically deploys to GitHub Pages when you push to the `master` branch.
